💻 UPDATE Statement 🌟 table lo unna data ni change cheyyalante update command ni use chestam
🌟 Lets change city of user Steve Rogers
🍁 Query:

UPDATE UserDetails

SET city = 'NewYork'

WHERE id = 2;

🍁 Outcome:
id firstname lastname city
2 Steve Rogers NewYork
🌟 WHERE clause yela help avtundi ante manam mention chesina condition ni meet ayithene Query lo operation ni perform chestundi
🌟 city ni NewYork ki change cheyyamanna yeppudaithe id = 2 avtundo
🌟 meeru where clause use cheyyakapote andari city NewYork ayipotundi
🌟 updating multiple values at a time
🍁 Query:

UPDATE UserDetails

SET firstname = 'Robert', lastname = 'Luna'

WHERE id = 5;

🍁 Outcome:
id firstname lastname city
5 Robert Luna Waverly

💻 Delete Statement 🌟 Table lo data ni delete cheyyalante delete command use chestam
🌟 truncate command undi kadha adhi use chedam ani meru anukovacchu kani only one person data ni delete cheyyalante ee delete command use cheyyali
🍁 Query:

DELETE FROM UserDetails

WHERE id = 5;

🍁 Outcome:
id firstname lastname city
1 Tony Stark Malibu
2 Steve Rogers NewYork
3 Bruce Banner Dayton
4 Natasha Romanoff Red Room
6 God Thor Asgard
🌟 id number 5 unna user data delete ayipotundi
🌟 table lo unna data antha delete avvalante, table name ni mention cheste chaalu
🍁 Query:

DELETE FROM UserDetails;

-- above query similar to below one

TRUNCATE TABLE UserDetails;

🍁 Outcome:
id firstname lastname city
🌟 Ayina delete enduku truncate use chesi where tho condition pedite saripotundi ane doubt ravochu
🌟 kani Truncate command does not support where clause anduke delete use chestam
🌟 table lo data antha teeseyalante truncate vaduko only rows ne delete cheyyalante delete command vaduko anthe.
