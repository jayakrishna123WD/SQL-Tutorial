💻 Data Control Language 🌟 nuv oka project meeda work chestunnav, yevado ne file access chesi delete cheseste yela untadi
🌟 anthe ga alane anipistundi ga. manaki ilanti situation lo help chesevade DCL -- Data Control Language
🌟 It helps in controlling access to information stored in a database
🌟 indulo two commands untay vatini use chesi yevadu mana files access cheyyochu, yevadiki mana files access undakudadho manam set cheskovacchu
🍁 GRANT Command:
🌟 syntax:

GRANT privilege_name on objectname to user;

🌟 privilege place lo ee commands ni use cheyyochu SELECT,UPDATE,DELETE,INSERT,ALTER,ALL according to your requirement
🌟 objectname ante table name
🌟 user place lo meeru yevarikaite privileges ni grant cheyyalanukuntunnaro valla name rastaru

GRANT SELECT, UPDATE ON Customers TO John;

🌟 John ane person ki Customers table lo data emundo choodagaladu and update kooda cheyyagaladu
🍁 REVOKE Command:
🌟 syntax:

same as GRANT kakapote dani place lo REVOKE ani pedtaru

REVOKE SELECT, UPDATE ON Customers TO John;

🌟 ippudu John gadu Customers table data choodaledu, update kooda cheyyaledu

💻 Transaction Control Language 🌟 ee topic antha malli ane word chuttune tirugutundi
🌟 by mistake oka file meeru delete chesesaru ventane undo ane block popup avtundi click cheste malli vachestundi delete ayina file
🌟 so, database lo data paina jarige anni Transactions ni control chesede ee TCL commands
🍁 COMMIT Command:
🌟 meeru table ki INSERT, DELETE or UPDATE operations chesetappudu avi temporary ga store avtay
🌟 permanent ga store avvali ant we use commit command
🍁 ROLLBACK Command:
🌟 anukokunda file delete chesi nappudu undo option undi ga adhe ROLLBACK ante
🍁 SAVEPOINT Command:
🌟 SAVEPOINT enti ante checkpoints laga manam save points ni pettukovacchu code madhyalo, ye point ki rollback avvalo ah point mention chestaru roll back pakkana
