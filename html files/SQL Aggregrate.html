💻 HAVING Clause 🌟 meeru HAVING clause telusukune mundu first HAVING vs WHERE telusukovali
🍁 HAVING vs WHERE:
HAVING WHERE
Possible to check condition on multiple rows at a time Possible to check condition for only one row at a time
We can use these Aggregate functions with this clause This clause can't support Aggregate functions
After execution of GROUP BY clause this clause will get executed GROUP BY clause will executed after execution of this clause
🍁 Customers Table:
Customerid name country
1 Tony USA
2 Steve UK
3 Bruce USA
4 Nat UAE
🍁 Query:

SELECT COUNT (Customerid) AS count, country

FROM Customers

GROUP BY country

HAVING COUNT (Customerid) < 2;

🍁 Customers Table:
count country
1 UK
1 UAE
